# Data Capturing Bot
This bot captures data from the excel and fill this information in the web form and desktop application parallely. It performs following:
- First It reads excel file with read range activity and save this information in data table variable.
- Opens a website in browser which contains the particular form to fill.
- Extracts data form excel row by row and type it into the form inputs.
- Adds the information and repeats the process for next row.
- Performs the similar activities in a provided desktop application 'MyCRM.exe'by launching it and filling out it's fields.
